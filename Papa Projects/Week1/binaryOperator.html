Binary Operator

What is Binary Operator

Binary operator is an operator that manipulats two operands and return a result. The operators are represented by special characters or Keywords that provide an easy way to compare numerical values and characters.

Binary operators are represented as follows:

Operand1 operator Operand2

Common Binary operators include the following:

Operator Operator Symbol Operator Type
Equal == Equality Operator
Not Equal != Equality Operator
Less Than < Relation Operator
Greater Than > Relation Operator
Less Than or Equal to <= Relation Operator
